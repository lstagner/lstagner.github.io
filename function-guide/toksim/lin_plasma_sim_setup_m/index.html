    lin_plasma_sim_setup.m¶

    Basic Information¶

      SYNTAX: lin_plasma_sim_setup

  PURPOSE:  Define configuration data structure for limited_system module.
    This function is called by invoking pcssp_bld_setup to provide structured
    data objects illustrating what fields need to be filled in for this module.

  INPUT:
    module_path   = path of module for which setup_data is being constructed

  OUTPUT:
       setup_data = setup data for this module
