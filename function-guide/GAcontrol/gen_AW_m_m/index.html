    gen_AW_m.m¶

    Basic Information¶

      SYNTAX:  AW_m=gen_AW_m(Ad,Bd,c)

  PURPOSE:  Generate antiwindup matrix, for bumpless transfer.

  INPUT:
    Ad, Bd = A,B matrices for A,B,C,D controller being switched to.
    c = number of powers of Ad to add for "wind-on".  If c=-1, then
        c is computed using gamma=max(real(eig(Ad))) (must have |gamma|<1)
        (optional, default is c=size(Ad,1))

  OUTPUT:
    AW_m = antiwindup matrix - apply to error at transition to "wind-on"
            the new controller
